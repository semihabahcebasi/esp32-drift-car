# Phase 1 — Wiring & Simulation

Bu aşamada ESP32 drift car projesinin temel bağlantıları Wokwi üzerinde simüle edilecektir.

## Bu aşamada yapılacaklar:
- ESP32 + L298N + Motor bağlantı planı
- Wokwi devresinin kurulması
- Motor pinlerinin test edilmesi
- Serial Monitor üzerinden sonuçların doğrulanması

## Notlar:
- Bu aşama tamamen simülasyon üzerindedir.
- Projenin sonunda gerçek ESP32 kartı ile aynı bağlantılar yeniden uygulanacak.
