# Phase 2 — Basic Motor Control

Bu aşamada ESP32 üzerinde motorları yazılım ile kontrol etmeyi öğreneceğiz.
Simülasyon yine Wokwi üzerinden yapılacak.

## Bu aşamada yapılacaklar:
- setMotorLeft()
- setMotorRight()
- ileri / geri / dur fonksiyonları
- Serial Monitor ile test

## Notlar:
- Bu aşamada henüz drift yok, sadece temel hareket.
- Kodlar `main.cpp` dosyasında tutulacaktır.
